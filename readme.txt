#Help

File=> New --- Restarts the Calculator
File=> Save ---Save the calculations

Edit=> Cut, Copy, Paste, Delete List --for cut, copy, paste and deletion

View=> Default, Light and Dark modes are available

Mode=> Standard And Scientific modes are available

Recent List can be activated by clicking on the clock like button

Basic operations Are done using Standard mode.
